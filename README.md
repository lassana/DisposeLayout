AdaptableLayout
=============

The layout that can arrange children automatically.

Code:
---
    AdaptableLayout layout = (AdaptableLayout) view.findViewById(R.id.layout);
    layout.addView(button1);
    layout.addView(button2);
    layout.addView(button3);
    //...
